Drill It

May 24th, 2014

Task

Create a 2D game that runs on a Playstation 2 running Linux.

Software used

Linux Playstation 2 development kit, Notepad++, make

Language used

C++, linux terminal commands

Description

The programme is based off of a flash game called motherload ( Link ). It is a simplified version of the game where the objective is to destroy as much of the ground as possible without running out of fuel. There is a refueling station (Red Square) where the player (Grey Square) can refuel the ship . The player is judged by the how quickly they can destroy the ground and how many ground pieces they destroy.

The source code for the project can be downloaded from this link: Drill It Source
